Welcome to your first coding activity! Before you begin coding away, let's first talk about what HTML is.

HTML stands for Hyper Text Markup Language and it describes the structure of a web page. Everything that you see on a web page is basically came from HTML, like this text!

You can create your own HTML page by creating an .html file and including the basic structure:

<!DOCTYPE html>
<html>
 <head>
  <title>Page Title</title>
 </head>
 <body>

 <h1>HTML is cool!</h1>
 <p>Space is cool!</p>

 </body>
</html>

You might have noticed that HTML uses these brackets <> to form elements, these are called tags. For each opening tag, you also need a closing one when you are done. The closing tag looks like this:

If you want to display text on the page, the popular tags to use are h1, h2, h3, h4, h5, h6, p, a The h tags are used for headings, the p tag is used for writing a paragraph, and the a tag is used to include links to other files or websites. You can link something by including href="some link" next to the initial a tag.
